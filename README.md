## 🧩 소개
이 프로젝트는 **테일즈위버**을 참고하여 **핵심 전투 시스템/챕터/UI 등**을 재현한 모작 프로젝트입니다.  
원작이 2D이기 때문에 3D 모작을 진행하면서 다음과 같은 변경사항을 고려하였습니다.

- ✅ 재현 목표: 테일즈위버 특유의 분위기, 전투시스템, 스토리텔링을 3D로 구현
- 🔧 한계: 테일즈위버 특화 3D에셋 부재로 인한 SD 캐릭터 에셋 구매
- ✨ 변경: 커스텀 에셋 추가, 언리얼 엔진 UMG를 사용해 원작 UI 재현
- 🎯 개선: 쉬운 타겟팅
